# Elder Scrolls Lore Reference — Index

This directory contains canonical lore references for mod accuracy checks.
Each file covers one topic. Search within a specific file for targeted lookups.

## Citation Key

Every entry in these files is tagged with one of the following status labels:

| Label | Meaning |
|---|---|
| `[CANON]` | Confirmed in official game text, developer statements, or in-game books. Cross-referenced with UESP. |
| `[CANON — UESP]` | Directly sourced from the UESP wiki, which aggregates official in-game sources. |
| `[CANON — IN-GAME BOOK]` | Sourced from a specific in-game book. Book title given in the entry. |
| `[CONTESTED]` | Multiple official sources contradict each other, or developer statements conflict with in-game text. Both positions are documented. |
| `[SOFT CANON]` | Appears in older games/guides that have since been superseded. Treated as true unless contradicted by newer material. |
| `[MOD CHOICE]` | A deliberate creative decision made for this mod that departs from or extends canon. Not an error. |
| `[SPECULATION]` | Widely accepted fan interpretation with no direct official source. Use cautiously. |

## Primary Sources (in order of reliability)

1. **In-game text** — dialogue, books, loading screens from the games themselves. Highest authority.
2. **UESP** (Unofficial Elder Scrolls Pages) — `https://en.uesp.net/` — the most rigorously sourced wiki. Preferred reference.
3. **The Imperial Library** — `https://www.imperial-library.info/` — compiles in-game books and some developer notes.
4. **Pocket Guide to the Empire (various editions)** — in-game books; 1st Edition is older/sometimes contradicted by 3rd Edition.
5. **Elder Scrolls Fandom Wiki** — `https://elderscrolls.fandom.com/` — broader coverage but less rigorous; useful for secondary checks.
6. **Developer interviews / MK posts** — statements from Michael Kirkbride and other developers. Treated as `[SOFT CANON]` or `[CONTESTED]` when they conflict with in-game text.
