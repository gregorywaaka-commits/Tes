# =============================================================
#  Vivec's Trial — Challenging the Living God
#  Namespace: vivec_trial
#
#  LORE:
#  Vivec is worshipped as one of the three living gods of the
#  Tribunal Temple.  But beneath that divine mask lies a deeply
#  complicated truth: Vivec was present at the murder of Nerevar,
#  the greatest Chimer general, at the Battle of Red Mountain
#  (1E 700).  The Tribunal's official history claims Nerevar died
#  fighting Dumac.  The Ashlanders — and the hidden texts of the
#  36 Lessons of Vivec — suggest otherwise: that the Tribunal
#  killed Nerevar to use the Tools of Kagrenac unopposed.
#
#  "I have a secret for you, Nerevar. I know what you are.
#   Do you know what I am?"  — 36 Lessons of Vivec, Sermon 36
#
#  Vivec never fully denies the accusation.  He layers it in
#  metaphor and paradox, but the admission is there for those who
#  read with clear eyes.  A Sixth House cultist (who reveres the
#  memory of Nerevar and Dagoth Ur) or the Nerevarine themselves
#  may choose to formally confront Vivec — not as a supplicant,
#  but as an accuser.
#
#  SEQUENCE:
#  ─────────
#  challenge_vivec (decision) → vivec_trial.000 → vivec_trial.010
#      → vivec_trial.020 (humbled) or vivec_trial.021 (fractured)
#
#  • vivec_trial.000  — "The Charges": the accusations are laid
#  • vivec_trial.010  — "The God Speaks": Vivec's response
#  • vivec_trial.020  — "A God Humbled": Vivec admits his role
#  • vivec_trial.021  — "The Tribunal Fractures": Vivec defends himself
#
#  KEY FLAGS:
#  ──────────
#  • vivec_trial_begun        — the trial is in progress
#  • vivec_charged_murder     — the charge of murdering Nerevar is laid
#  • vivec_charged_betrayal   — the charge of betraying Dunmer heritage
#  • vivec_charged_false_god  — the charge of false divine ascension
#  • vivec_humbled            — outcome: Vivec acknowledges the truth
#
#  EXTERNAL DEPENDENCIES:
#  ─────────────────────
#  • dunmer_culture       = EK2 culture identifier
#  • sixth_house_cultist  = trait in dagoth_ur_traits.txt
#  • nerevarine_marked    = trait in dagoth_ur_traits.txt
#  • vivec_blessed        = trait in tribunal_traits.txt
#  • dagoth_ur_awakened   = scripted trigger in lore_races_triggers.txt
# =============================================================

# =============================================================
#  LORE NOTES — ALL THREE TRIBUNAL TRIALS
#
#  Each god responds in a way consistent with their character:
#
#  VIVEC       — the warrior-poet.  He engages with his own
#                contradictions.  He will not rage or deny outright.
#                He layers truth in paradox and may, if pressed, allow
#                something close to an admission.  The most nuanced trial.
#
#  ALMALEXIA   — the mother-protector.  She does not philosophise.
#                She regards the accusation as a personal betrayal and
#                responds with fury.  Pressing her is genuinely dangerous —
#                but a ruler who holds firm exposes that a goddess who
#                cannot bear judgment is no god at all.
#
#  SOTHA SIL   — the Clockwork God.  He is a recluse who barely
#                interacts with mortals.  He does not deny his complicity;
#                he simply does not accept that mortal justice is the
#                correct frame.  His trial is the most unsettling: he
#                might answer your charges fully and still leave you with
#                nothing that feels like resolution.  Lore note: Sotha
#                Sil retreated to his Clockwork City partly out of
#                genuine shame over Nerevar's murder — scholars debate
#                whether this constitutes remorse or avoidance.
# =============================================================


# =============================================================
#  CROSS-REFERENCE INDEX  —  events/vivec_trial_events.txt

#      → vivec_blessed
# =============================================================

# ---------------------------------------------------------------
# vivec_trial.000  The Charges — Confronting Vivec
# The character steps forward to formally accuse Vivec.
# Three possible charges: the murder of Nerevar, the betrayal
# of the old Dunmer ways, or the theft of false godhood.
# All are lore-grounded accusations; the charge shapes how
# Vivec defends himself in the next event.
# ---------------------------------------------------------------
vivec_trial.000 = {
    type = character_event
    title = vivec_trial.000.t
    desc = vivec_trial.000.desc
    theme = faith

    option = {
        name = vivec_trial.000.a   # "You murdered Nerevar — our greatest general, our champion."
        trigger = {
            has_trait = nerevarine_marked
        }
        add_character_flag = { flag = vivec_trial_begun }
        add_character_flag = { flag = vivec_charged_murder }
        add_piety = 150          # Invoking the Nerevarine prophecy openly draws Azura's favour
        add_stress = 25
        trigger_event = { id = vivec_trial.010 days = 10 }
    }

    option = {
        name = vivec_trial.000.b   # "You betrayed the Chimer's true path and the Daedra who made us."
        trigger = {
            has_trait = sixth_house_cultist
        }
        add_character_flag = { flag = vivec_trial_begun }
        add_character_flag = { flag = vivec_charged_betrayal }
        add_intrigue = 1         # The Sixth House's accusation is politically sharp
        add_stress = 20
        trigger_event = { id = vivec_trial.010 days = 10 }
    }

    option = {
        name = vivec_trial.000.c   # "You are no god. You stole divine power that was not yours to take."

        add_character_flag = { flag = vivec_charged_false_god }
        add_piety = 100
        add_stress = 30
        trigger_event = { id = vivec_trial.010 days = 10 }
    }
}

# ---------------------------------------------------------------
# vivec_trial.010  The God Speaks — Vivec's Defense
# Vivec responds to the charges.  He does not rage or deny
# outright — that is not his nature.  He layers truth in paradox.
# The character must decide whether to press harder (seeking a
# true admission and humbling the god) or to accept the
# deflection (granting Vivec his dignity but gaining nothing).
# ---------------------------------------------------------------

    type = character_event
    title = vivec_trial.010.t
    desc = vivec_trial.010.desc
    theme = faith

    trigger = {
        has_character_flag = vivec_trial_begun
    }

    option = {
        name = vivec_trial.010.a   # "Your poetry does not answer my charge. Speak plainly, god."
        trigger = {
            learning >= 12   # Cutting through Vivec's metaphor requires learned insight
        }
        add_stress = 20
        add_learning = 1
        trigger_event = { id = vivec_trial.020 days = 10 }    # Vivec is pressed — and humbled
    }

    option = {
        name = vivec_trial.010.b   # "You speak in circles to obscure the blood on your hands."
        trigger = {
            martial >= 12
        }
        add_stress = 15
        add_prestige = 100
        trigger_event = { id = vivec_trial.020 days = 10 }    # Pressure through will — also succeeds
    }

    option = {
        name = vivec_trial.010.c   # "Perhaps I misunderstood. The gods are beyond mortal judgment."
        add_stress = -10           # Accepting the deflection relieves stress — but gains nothing
        add_piety = 50             # The Tribunal Temple approves of your deference
        remove_character_flag = vivec_trial_begun
        remove_character_flag = vivec_charged_murder
        remove_character_flag = vivec_charged_betrayal
        remove_character_flag = vivec_charged_false_god
        trigger_event = { id = vivec_trial.021 days = 10 }    # Vivec "wins" — the fracture does not come
    }
}

# Lore note: Vivec is the only Tribunal member who engages

# rage; Sotha Sil would close his door.  Only Vivec allows this.

vivec_trial.020 = {
    type = character_event
    title = vivec_trial.020.t
    desc = vivec_trial.020.desc
    theme = faith

    trigger = {
        has_character_flag = vivec_trial_begun
    }

    option = {
        name = vivec_trial.020.a   # "I did not expect that. But truth has its own gravity."
        add_character_flag = { flag = vivec_humbled }
        add_piety    = 500          # Azura's prophecy acknowledged — enormous spiritual reward
        add_prestige = 800          # The Nerevarine who cornered a god earns lasting renown
        add_stress   = -20          # Resolution, however painful, relieves the weight
        add_character_modifier = {
            modifier = vivec_trial_victor_modifier

        }
        remove_character_flag = vivec_trial_begun
        remove_character_flag = vivec_charged_murder
        remove_character_flag = vivec_charged_betrayal
        remove_character_flag = vivec_charged_false_god
    }
}

# =============================================================

#  Namespace: almalexia_trial
#
#  LORE:
#  Almalexia — Ayem — is the most politically aggressive of the

#  conspirator as Vivec and Sotha Sil.
#

#  philosophical puzzle.  She experiences it as a personal
#  affront: how dare a mortal question the Mother of Morrowind?
#  Her response is fury — which, paradoxically, is exactly what
#  an accuser needs.  A god who cannot bear judgment is no god
#  at all.  That fury, visible to all, is the evidence.
#
#  Pressing the charge against Almalexia requires more raw
#  courage than Vivec's trial — she will not deflect gently.
#
#  SEQUENCE:
#  ─────────
#  challenge_almalexia (decision) → almalexia_trial.000 →
#      almalexia_trial.010 → almalexia_trial.020 (exposed)

#  • almalexia_trial_begun     — the trial is in progress

# =============================================================
#  SOTHA SIL'S TRIAL — Confronting the Clockwork God
#  Namespace: sotha_sil_trial
#
#  LORE:
#  Sotha Sil retreated to his Clockwork City almost immediately
#  after the Battle of Red Mountain and rarely emerged.  In lore,
#  scholars believe his retreat was partly motivated by genuine
#  shame over Nerevar's murder — though whether that constitutes
#  remorse or mere discomfort is debated.
#
#  Unlike Vivec (who philosophises) or Almalexia (who rages),
#  Sotha Sil is indifferent — not callously, but cosmically.
#  He has stepped back from mortal affairs to work on what he
#  believes is the real problem: the fundamental nature of reality.
#  Mortal justice, in his view, is not the correct frame for
#  what happened.  He might acknowledge the murder fully — and
#  then ask whether that acknowledgment changes anything.
#
#  The charge reaches him via a message to the Clockwork City.
#  Remarkably, he responds — because Sotha Sil is, at his core,
#  an intellectual who respects a question genuinely asked.
#
#  Requires learning >= 16 to formulate a charge that can reach
#  him across the conceptual distance of the Clockwork City.
#
#  SEQUENCE:
#  ─────────
#  challenge_sotha_sil (decision) → sotha_sil_trial.000 →
#      sotha_sil_trial.010 → sotha_sil_trial.020 (answered)
#                          or sotha_sil_trial.021 (dismissed)
#
#  KEY FLAGS:
#  ──────────
#  • sotha_sil_trial_begun  — the challenge has been issued
#  • sotha_sil_charged      — the charges have been laid
#  • sotha_sil_answered     — Sotha Sil answered the charge fully
# =============================================================

namespace = sotha_sil_trial
